# TubeLuLu 서비스 소개서

LUMIA 시절 인플루언서 제안 슬라이드(`../index.html`)를 TubeLuLu 리브랜딩과
현행 제품 사양에 맞춰 전면 개편한 서비스 소개서입니다.

## 보는 방법

`index.html`을 브라우저에서 열면 됩니다. 빌드 불필요.

- 이동: `←` `→` 방향키, 스페이스, 터치 스와이프, 하단 도트 클릭
- `Home` / `End`: 첫/마지막 슬라이드
- 인쇄(`Cmd+P`): 슬라이드당 1페이지로 출력 (가로 방향 권장)
- 어느 해상도에서든 16:9 (1280×720 기준)로 자동 스케일링

## 구성 (13장)

표지 → 고민 → 선언 → 시장 → 기회 → 서비스 화면 → 팬 심리 → 시스템 →
수익 시나리오 → 팬 유입 → 역할 분담 → 정식 서비스 → 마무리

# TubeLuLu 서비스 소개서

LUMIA 시절 인플루언서 제안 슬라이드(`../index.html`)를 TubeLuLu 브랜드와
현행 제품 사양에 맞춰 전면 개편한 서비스 소개서입니다.

- **`index.html`** — 현행본. "Ink & Ice" 다크 시네마틱 에디토리얼 디자인
  (딥 네이비 잉크 필드 · 아이스 블루 액센트 · 고스트 넘버 · 헤어라인 그리드 ·
  초대형 타이포/수치 · 세리프 스테이트먼트 · 화이트 제품 목업)
- **`v1.html`** — 이전 버전. 라이트 카드 기반 디자인 (참고용 보존)

## 보는 방법

`index.html`을 브라우저에서 열면 됩니다. 빌드 불필요.

- 이동: `←` `→` 방향키, 스페이스, 터치 스와이프, 하단 틱 클릭
- `Home` / `End`: 첫/마지막 슬라이드
- 인쇄(`Cmd+P`): 슬라이드당 1페이지 출력 (배경 그래픽 인쇄 옵션 권장)
- 어느 해상도에서든 16:9 (1280×720 기준)로 자동 스케일링

## 구성 (13장)

표지 → 고민 → 선언 → 시장 → 기회 → 서비스 화면 → 팬 심리 → 시스템 →
수익 시나리오 → 팬 유입 → 역할 분담 → 왜 지금 → 마무리
